# PastelBoard

![PastelBoard icon](assets/icon.png)

PastelBoard 是一款仅面向 Android 的远程蓝牙虚拟触控板与键盘应用，目标是让手机横屏后像一台粉紫色、软萌风格的笔记本电脑键盘，同时可以通过蓝牙 HID 向已配对设备发送真实键位与指针输入。

## 默认视觉方向

- 默认主色：粉色、纯紫色，以及贴近图标的粉紫渐变。
- 主题系统：基于 Material You，整体使用圆角、透明毛玻璃、柔和阴影和软糖感键帽。
- 字体：内置「霞鹜文楷」，风格更柔和，并且支持中文显示。
- 键盘形态：按真实笔记本电脑键盘布局设计，而不是普通输入法键盘。
- 按键反馈：点击磨砂玻璃键帽会在键帽上方出现放大的原创金色烟花动画，并播放清脆 Flick 感原创按键音。
- 交互菜单：主页与控制页使用侧边展开式菜单，方便进入设置和键盘测试页。

## 核心交互

- 启动后先进入设备选择页，让用户选择要连接的蓝牙设备。
- 设备选择页左上角提供菜单入口，菜单内包含设置。
- 连接设备后强制横屏，顶部保留固定切换栏，用于切换「触控板」和「键盘」。
- 设置页支持切换默认颜色配置，也支持 RGB 调色板和 `#RRGGBB` / `#AARRGGBB` 十六进制颜色输入。
- 设置页支持选择本地图片作为应用背景，并调整图片透明度；颜色、背景和按键音设置会保存到本地。
- 设置页支持开启即横屏、自定义按键音文件、自定义键帽颜色。
- 键盘测试页可以在不发送蓝牙按键的情况下测试键帽颜色、金色烟花动画和按键音。

## v0.2.2 更新重点

- 强化键帽磨砂玻璃效果：加入高光、雾面颗粒、柔化光晕和更明显的玻璃边缘。
- 修复按键音稳定性：优先使用 `SoundPool`，未加载或播放失败时自动切到 `MediaPlayer`。
- 按键音现在走媒体音量，并保留系统点击音作为最后兜底，减少测试页无声情况。

## v0.2.1 更新重点

- 精简主页：去掉图标下方介绍卡和功能标签，让蓝牙设备列表更靠上。
- 优化蓝牙设备项：设备名与地址单行省略，连接按钮更紧凑，避免窄屏换行挤压。
- 优化键盘反馈：键帽改为磨砂玻璃样式，金色烟花显示在键帽上方，尺寸放大并持续 1.5 秒。
- 修复按键音：提前注册 SoundPool 加载监听，自定义音频加载慢时会在加载完成后补播。

## v0.2.0 更新重点

- 新增侧边展开式菜单，替代旧下拉菜单。
- 新增键盘测试页，适合调试按键效果和声音。
- 新增自定义按键音、本地音频选择、恢复默认按键音。
- 新增键帽颜色单独调节，支持 RGB 滑块和十六进制输入。
- 新增「开启即横屏」，并放宽界面间距、增强文字对比、放大横屏键盘区域。

## 构建

本仓库配置了 GitHub Actions：推送到 `main` 或创建 Pull Request 后，会在 GitHub 上执行 Android Debug APK 构建并上传构建产物。

## 当前状态

项目已初始化为 Android Kotlin + Jetpack Compose 工程，包含应用图标、Material You 粉紫主题、设备选择页、设置页、触控板界面、笔记本式虚拟键盘界面、毛玻璃背景系统、按键动效与音效，以及蓝牙 HID 报告发送的基础结构。
